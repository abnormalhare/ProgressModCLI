## Installation

Built on Python 3.

Dependencies: ```rich```

1. Download the appropriate zip file for your system and extract.
2. Open your terminal/command prompt inside the extracted contents.
3. Make sure the depenencies have been met.
4. Simply do ```python boot.py```
